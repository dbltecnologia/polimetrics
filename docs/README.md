# PoliMetrics: A Arquitetura da Inteligência Política

Bem-vindo ao repositório central de documentação da **PoliMetrics** (anteriormente Mapa Político / Estratégia Política Inteligente). Este diretório atua como o **cérebro** do projeto, contendo as informações necessárias para as equipes de **Marketing, Comercial, Estratégia e Operação**.

A PoliMetrics foi purificada e reestruturada para focar estritamente em **6 Pilares** de conversão eleitoral e gestão de mandato, removendo ruídos de antigas funcionalidades (gamificação, rankings vazios) que não traziam dados concretos para a mesa de decisão.

Se você precisa vender a plataforma, entender como ela resolve problemas reais através do Gabinete Virtual e Mapas Interativos, ou treinar novos usuários, você está no lugar certo.

## Índice de Documentação

1. **[Visão de Negócio (Business & Marketing)](./01-VISAO-BUSINESS.md)**
   - O pitch comercial, proposta de valor (ROI), público-alvo (B2B, B2G) e o argumento de conversão direta. Leia isto se você vai apresentar o sistema para um candidato, partido político ou estrategista-chefe.

2. **[Funcionalidades Chave (Os 6 Pilares)](./02-FUNCIONALIDADES-CHAVE.md)**
   - O que o sistema faz? Um resumo executivo dos 6 módulos logísticos do sistema, detalhando o que cada Pilar resolve (Inteligência Geográfica, Ponte Permanente pelo Gabinete Virtual, Histórico Eleitoral).

3. **[Casos de Uso (Cenários Práticos)](./03-CASOS-DE-USO.md)**
   - Exemplos reais de como a plataforma resolve as dores agudas de uma eleição ou mandato, tais como o abandono de base pós-eleição, a perda de engajamento do líder de bairro ou a cegueira geográfica durante o planejamento operacional.

4. **[Manual do Usuário (Operacional Rápido)](./04-MANUAL-DO-USUARIO.md)**
   - Um guia robusto e dinâmico que destrincha a Interface visual, dividindo os atalhos e as responsabilidades do **Administrador** (Candidato/Coordenador no Dashboard Web) e do **Líder** (Cabo Eleitoral/Apoiador Principal atuando "Mobile-first" nas ruas).
